##  Getting Started

### 1. Install FlutterFire
```bash
dart pub global activate flutterfire_cli
```

### 2. Configure Firebase
```bash
flutterfire configure
```
- Select your Firebase project
- Enable Firestore and Storage
- Select all platforms

### 3. Create Firestore Collections
- **henna_categories** - Category documents
- **henna_images** - Individual design documents
(See FIREBASE_SETUP.md for structure)

### 4. Upload Design Images
- Use Firebase Storage
- Update Firestore document URLs

### 5. Connect to Home Screen
Add navigation button to HomeScreen:
```dart
ElevatedButton(
  onPressed: () => Navigator.pushNamed(context, '/categories'),
  child: const Text('Browse Firebase Designs'),
)
```

### 6. Run the App
```bash
flutter pub get
flutter run
```
